# 📚 AI Research Paper Reader (RAG System)

An AI-powered system that allows users to upload research papers and ask questions from them using Retrieval-Augmented Generation (RAG).

---

## 🚀 Features
- Upload PDF research papers
- Automatic text chunking
- FAISS-based semantic search
- LLM-powered answers (Groq API)
- Streamlit UI (ChatGPT-like interface)

---

## 🧠 Tech Stack
- Python
- Streamlit
- FAISS
- Sentence Transformers
- Groq LLM API
- PyPDF
